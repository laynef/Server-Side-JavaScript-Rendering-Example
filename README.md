## How to test
--> Must have Redis Server installed for backend caching<br>
`mkdir homebrew && curl -L https://github.com/Homebrew/brew/tarball/master | tar xz --strip 1 -C homebrew`<br>
`brew install redis`<br>

### To start
`npm install`<br>
`npm start`<br>
